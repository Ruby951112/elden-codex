## How to use these

1. The AI pipeline already generates relevant `keywords:` arrays per boss. After running, spot-check the TOP 30 above to verify quality.

2. For each TOP 30 boss, manually edit the `description` field in `data/bosses.json` so it includes the primary keyword. Example:

```json
{
  "slug": "malenia-blade-of-miquella",
  "summary_en": "How to beat Malenia: Waterfowl Dance has 3 distance-based dodges...",
}
```

3. Internal linking: Make TOP 30 boss pages link to each other where contextually relevant. The "Related Bosses" section in MDX should reference 2-3 thematically connected bosses.

4. **Search Console after launch**: Submit `eldencodex.com/sitemap.xml` and watch which queries actually drive traffic. Adjust descriptions monthly based on real data, not these estimates.
