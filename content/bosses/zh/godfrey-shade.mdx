## 阶段策略

葛弗雷影是游戏对你的一次阶段性检测——在进入王城罗德尔核心区域之前,开发者希望确认你已经掌握了延迟攻击的节奏。这场战斗只有单阶段,但葛弗雷的每一招都充满欺骗性的前摇停顿。如果你还在凭肌肉记忆翻滚,这里就是你的墙。

葛弗雷影出现在王城罗德尔东首都城壁赐福附近,穿过黄金树根须的通道后,他会站在通往蒙葛特的必经之路上。这是一场剧情意义重大的战斗——葛弗雷是初代艾尔登之王,也是葛孚雷战神的"黄金时代"投影。游戏在此告诉你:你即将挑战的不是普通君王,而是传奇本身。

> **关键提示:** 葛弗雷影的核心机制是**延迟踏地震荡**。他会抬起脚悬停 1-2 秒,然后重踏地面释放巨大范围的冲击波。大多数玩家在前三次尝试中都会死于过早翻滚。正确应对是:**盯住他的脚,只在脚落地的瞬间翻滚**。

### 单阶段——王者威压

葛弗雷影挥舞一柄金色战斧,攻击模式分为三类:

- **斧头横扫与竖劈**——标准近战连段,前摇有明显停顿
- **踏地震荡**——范围型控制技能,整个竞技场几乎无死角
- **抓取投技**——单一最高伤害技能,但前摇极长且可躲

战斗节奏是**引诱-惩罚**循环。葛弗雷的攻击前摇很长,但判定非常精准。他不会给你连续输出的机会——每次攻击窗口只够你打一到两段R1。贪刀会被他的超级护甲打断,随后接踏地或抓取。

血量无阈值变化。葛弗雷影从开局到结束都使用相同招式池,但血量越低,他使用踏地和抓取的频率越高。保持耐心,这场战斗是马拉松而非短跑。

## 招式与应对

| 招式 | 应对 |
|------|------|
| 单次踏地震荡 | 他抬起右脚悬停。**等脚落地后立刻翻滚**。过早翻滚会被冲击波扫中。 |
| 双重踏地震荡 | 连续两次踏地,第二次范围更大。第一次翻滚后立刻准备第二次。 |
| 战斧横扫(单段) | 向他挥斧方向翻入。横扫判定持续时间短,翻入即可到他侧身。 |
| 战斧三连竖劈 | 第一段和第二段向左侧翻。第三段会有停顿,延后0.5秒再翻。 |
| 跳跃下劈 | 他跳起后悬停。**盯住斧头,斧头下落瞬间向前翻入**。落地冲击波范围小。 |
| 抓取投技 | 他张开双臂缓慢走近。向左右任意方向翻滚即可完全躲开。被抓后直接扣70%血量。 |
| 战吼震退 | 罕见招式,他会怒吼并向前推出冲击波。保持中距离即可避免,近身时向后连翻两次。 |

## 推荐配装

葛弗雷影的弱点是**出血**。他没有绯红腐败抗性,出血触发可削减15%最大血量,且会打断他的攻击前摇。

### 近战流派

- **武器:** **血河**、**血犬步法的大刀**、**莫格温圣枪**、或任何带有**出血战灰的打刀**。双持打刀流在此战中表现极佳。
- **战灰:** **切腹**(打刀系列)或**猎犬步法**(快速连击触发出血)。避免使用慢速重武器——葛弗雷的攻击窗口不够你挥完。
- **护符:**
  - **深红琥珀勋章+2**(HP提升,应对踏地误判)
  - **绿龟护符**(耐力回复,保证连续翻滚)
  - **腐败龙印护符**(提升强韧,避免被轻击打断)
  - **亚历山大的碎片**(提升战技伤害,适配血河流派)

### 法师流派

- **法术:** **彗星亚兹勒**或**洛蕾塔的大弓**。葛弗雷移动缓慢,远程蓄力法术命中率高。
- **护符:** **魔力蝎符咒**、**拉达冈的肖像**(加快施法速度)。
- **战术:** 召唤**拟态之泪**或**孤狼骨灰+10**分担仇恨,你在远距离蓄力输出。

### 遗灰选择

- **拟态之泪+10**——最稳定方案。拟态会复制你的装备和血量,能抗下两到三次踏地震荡。
- **黑刀缇切+10**——高闪避能力,能持续输出并触发HP削减debuff(降低葛弗雷最大血量)。
- **孤狼骨灰+10**——三只狼分散目标优先级,给你更多输出时间。

> **战术提示:** 葛弗雷影没有远程攻击。如果你保持中远距离,他只会使用踏地震荡或慢速走近。这是法师和弓箭手的天然优势——引诱踏地,翻滚后立刻反击,再次拉开距离。

## 延迟踏地的核心理解

葛弗雷的踏地震荡是许多玩家卡关的原因。它违反了黑暗之魂系列的"看到抬手就翻滚"直觉。正确应对需要新的反射弧:

1. **视觉提示:** 他抬起右脚,脚悬停在空中。
2. **音频提示:** 他会轻微吸气(耳机玩家能听到)。
3. **翻滚时机:** 只在脚接触地面的瞬间按翻滚键。

训练方法:前五次尝试**只专注于躲避踏地**,不要输出。等你能连续躲开十次踏地后,再开始反击。这个训练会让后续战斗轻松三倍。

如果你实在无法掌握延迟翻滚,备选方案是:**装备轻载装备(负重低于30%)**,提升翻滚距离。轻载翻滚能在过早触发时仍然清出冲击波范围。

## 剧情联系与后续

击败葛弗雷影后,前方通道会打开,直通女王寝室的雾门——蒙葛特,王城的恩惠就在那里等你。葛弗雷影的存在是一次预演:真正的葛弗雷(霍拉·露)在艾尔登王座前等着你,那场战斗会用尽你在此学到的一切。

葛弗雷影掉落**80000卢恩**——这是游戏在告诉你"你已经进入后期"。在挑战蒙葛特之前,确保你的武器已升级到+20(普通)或+8(传说),且赐福瓶已升级到+10以上。

> *葛弗雷不会说话,但他的每一斧都在说:慢下来,读懂节奏,成为艾尔登之王不是靠冲刺完成的。*

## 相关Boss

打完葛弗雷影后,下一站是**王城的恩惠,蒙葛特**——又一位延迟攻击大师,但他的连段比葛弗雷复杂三倍。如果你在此战中掌握了延迟翻滚,蒙葛特会变得相当可控。在游戏后期,你还会在艾尔登王座前再次遇到**初代艾尔登之王,葛弗雷(霍拉·露)**——那是他的完整形态,拥有第二阶段和全新招式池。
