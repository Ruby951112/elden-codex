## 阶段策略

波雷阿利斯是巨山山顶地区最标志性的野外龙战。与地下墓地那些困在狭小空间的龙不同,这场战斗在凛冬湖的开阔冰面上展开——开发者的设计意图非常明确:**骑上托雷特,像骑士一样战斗**。

核心策略只有一条:**永远顺时针绕着他的后腿跑**。波雷阿利斯的所有吐息攻击都向前或向左发射——顺时针跑位会让你始终处于他的盲区。不要贪刀,不要在他面前停留。打两下后腿,立刻冲刺拉开距离,等下一轮冰息结束再折返。

这场战斗的难度取决于你的耐心。如果你试图徒步硬拼,波雷阿利斯会用锥形冰息把你冻成冰雕。如果你骑马但逆时针跑,他的横扫冰息会直接命中你。**顺时针**是唯一正解。

> **关键提示:** 波雷阿利斯在第一次受到伤害后会立刻起飞并移动到湖的另一端。不要追——等他落地后骑马靠近。这是他的固定行为模式,每次血量下降到特定阈值(约 75%、50%、25%)都会触发。

### 龙战的节奏

所有龙战都遵循同一套节奏:

1. **接近阶段** — 龙在远处吐息。骑马冲刺接近,利用托雷特的双段跳规避地面冰霜。
2. **攻击窗口** — 到达后腿侧面,连续攻击 2-3 次。
3. **脱离阶段** — 龙抬头准备下一轮吐息。立刻冲刺离开攻击范围。
4. **等待阶段** — 保持中距离绕圈,观察他的下一个动作。

波雷阿利斯的特殊之处在于他的冰息攻击**积累冻伤状态**。冻伤槽满后你会瞬间损失大量血量并短暂僵直——在龙战中僵直 = 死亡。装备**抗寒护符**能显著减缓冻伤积累速度。

## 招式与应对

| 招式 | 应对 |
|------|------|
| 锥形冰息(前方) | 骑马顺时针全速冲刺。不要翻滚,托雷特的移速能跑出锥形范围。 |
| 横扫冰息(左至右) | 他会转头从左向右喷吐。顺时针跑位的你在他右后方,完全避开。 |
| 地面冰霜喷吐 | 低头向地面喷吐,扩散冰霜地面效果。双段跳离开,冰霜持续 5 秒。 |
| 起飞后俯冲 | 他飞向空中后会俯冲喷吐一次。看到他起飞立刻横向冲刺 90 度转向。 |
| 尾扫(近战) | 如果你站在他屁股正后方太久,他会用尾巴横扫。贴近侧后腿而非正后方。 |
| 翅膀拍击 | 他向两侧张开翅膀用力拍地。非常罕见,只在你长时间卡在他侧面时触发。向后拉开 5 米。 |

波雷阿利斯没有复杂的连段——他的威胁全部来自冰息的持续伤害和范围。学会识别他抬头的预兆,每次看到抬头就立刻冲刺,这场战斗会变得极度简单。

## 推荐配装

这场战斗的装备选择围绕两个核心:火焰伤害 + 抗冻伤。

- **武器:** 任何附带火焰属性或火焰战灰的武器。推荐**火焰战技:狮子斩**、**火焰赐予武器**、或**火焰涂油脂**。龙类对火焰的弱点极其显著——一次火焰 L2 能打出相当于三次普通 R1 的伤害。
- **护符:**
  - **抗寒护符+1**(在凛冬湖东侧的雪山洞窟获取)— 必带。能将冻伤积累减缓 40%。
  - **龙印大盾护符**(物理减伤)— 虽然波雷阿利斯主要是魔法伤害,但他的翅膀和尾巴是物理判定。
  - **深红琥珀勋章+2**(HP 提升)— 增加冻伤触发后的存活空间。
- **圣杯瓶配置:**
  - **火焰强化圣杯泪**(增加火焰伤害 20%)
  - **耐力回复圣杯泪**(托雷特的冲刺消耗你的耐力,回复速度决定你的走位频率)
- **道具:** **解冻药**×5。冻伤槽快满时提前喝,不要等触发后再喝——触发后的僵直窗口通常来不及。

如果你是法师流派,**坎恩的岩浆**或**魔法剑波纹斩(火焰注魔版)**都是优秀选择。远程施法在龙战中风险极高——波雷阿利斯的锁定距离很远,你施法时他可能已经在吐息了。

### 托雷特战术细节

许多新手在龙战中低估托雷特的重要性。以下是关键操作:

- **永远在移动中攻击**。不要停下来挥剑——托雷特的移动攻击判定和站立攻击完全相同,但你保持了机动性。
- **双段跳用于规避,不用于接近**。托雷特的双段跳有 0.5 秒的无敌帧——利用它穿过地面冰霜而非绕开。
- **喝瓶前拉开到安全距离**。托雷特上喝瓶的动作比徒步慢 0.3 秒,且无法取消。只在波雷阿利斯刚起飞或刚落地时喝。
- **召唤托雷特的音效是预警信号**。如果托雷特被击杀,你会听到特殊音效——立刻吹口哨重新召唤。托雷特的二次召唤需要消耗一瓶绯红圣杯瓶,但比你徒步逃命便宜得多。

## 战场环境

凛冬湖是一片完全平坦的冰面——没有掩体,没有高低差,无处可躲。这是刻意的设计:开发者希望你依赖托雷特的机动性而非地形优势。

波雷阿利斯会在战斗过程中多次起飞并移动到湖的不同位置。每次移动后他会落地并立刻释放一次大范围冰息——**不要在他落地瞬间靠近**。等冰息结束,再骑马冲刺到他的后腿侧。

湖面的冰霜地面效果会持续积累冻伤。如果你发现自己被冰霜围困,立刻双段跳离开——冻伤触发的瞬间伤害足以秒杀 40 活力以下的角色。

## 关于难度评级

波雷阿利斯的 3/5 难度评级非常准确——前提是你骑马。如果你徒步,这场战斗会飙升到 5/5。开发者通过这场战斗传达的核心教学是:**不是所有战斗都应该徒步硬拼**。

从设计哲学来说,波雷阿利斯是对传统"翻滚+走位"思维的挑战。魂系列老玩家往往会抗拒骑马战斗,因为托雷特的操作手感和徒步完全不同。但巨山山顶地区的所有野外龙战(波雷阿利斯、格兰萨克斯、凋零龙)都在暗示同一件事:学会骑马,否则寸步难行。

> *龙不是用来翻滚的,而是用来追逐的。* 把这场战斗当成骑士竞速而非格斗,胜利会自然而然到来。

## 相关 Boss

击败波雷阿利斯后掉落**龙心脏**和 50000 卢恩——这是巨山山顶地区最慷慨的单次卢恩奖励之一。龙心脏可以在龙餐教堂或玛尼斯卡教堂兑换龙系祷告,其中**波雷阿利斯的雾息**是对冰属性流派极有价值的 AOE 祷告。

如果你喜欢龙战,接下来推荐去王城罗德尔外的**格兰萨克斯**(雷电龙)——他的机制和波雷阿利斯类似,但速度更快,惩罚失误的力度更大。
