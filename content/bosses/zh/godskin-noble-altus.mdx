## 阶段策略

神肌贵族是艾格雷神殿的守门人,也是你在通往火山官邸路上的第一道真正考验。这场战斗的核心在于**站位管理**与**耐心**。神肌贵族的攻击距离极长,判定宽松,但几乎每一招都有明显的预兆窗口。你不需要反应快——你需要看懂他的身体语言。

这是一场单阶段战斗,但难度曲线在血量低于 40% 时陡增。神肌贵族会更频繁地使用**膨胀翻滚**,且连段之间的停顿时间缩短。前期不要贪刀攒血瓶——后期会需要每一瓶。

> **关键提示:** 神肌贵族对**削韧**和**黑焰**异常脆弱。跳跃重击流和黑焰武器可以在 90 秒内结束战斗。如果你有黑焰刀或黑焰之刃战灰,这场战斗会简单一个难度级别。

### 核心机制——膨胀翻滚

神肌贵族最具威胁的招式是**膨胀翻滚**。他会吸气膨胀,身体变成一个巨大的肉球,然后向你翻滚过来。这一招的判定范围比视觉效果大 30%,且持续时间长达 6-8 秒。

**正确应对方法:**

1. 看到他开始膨胀时,立刻**垂直他的方向冲刺**。不要翻滚,不要向后退——冲刺。
2. 当他开始翻滚的瞬间,向**他的侧面**翻滚一次。翻滚方向与翻滚路径平行,而非垂直穿过。
3. 他会翻滚 2-3 圈,然后停下来恢复人形。停下的瞬间是你最大的输出窗口——跳跃重击或蓄力 R2。

错误做法:向后翻滚会让你被第二圈碾过;站在原地会被直接压扁;向前翻入翻滚路径会被判定到第二圈。

### 削韧策略

神肌贵族的削韧阈值相对较低。连续的跳跃重击、战技或重型武器攻击能在 4-6 次命中内打出硬直。硬直后的致命一击伤害约为 1200-1500,相当于他 25% 的血量。

推荐削韧武器:

- **巨剑/特大剑**:每次跳跃重击造成约 180 削韧。
- **战锤/战灰**:地震波、猛冲等战技单次削韧值超过 200。
- **战灰组合**:狮子斩、巨兽咆哮在命中后能直接触发削韧。

如果你使用轻武器(打刀、直剑),削韧不是优先选项。专注闪避反击即可。

## 招式与应对

| 招式 | 应对 |
|------|------|
| 膨胀翻滚(2-3 圈) | 垂直冲刺,侧向翻滚。停止瞬间跳跃重击。 |
| 刺剑三连突刺 | 向他左侧(你的右手边)翻滚。第三段后有 1.5 秒窗口。 |
| 横扫刺剑 | 向前翻入他身体。横扫判定在身前 180°,贴身无判定。 |
| 黑焰火球 | 向左右平移走位。火球速度慢但追踪强,翻滚太早会被追上。 |
| 旋转刺剑斩 | 他会旋转 360° 挥剑。向后拉开距离,绝不翻入。 |
| 腹部撕裂(近身) | 他会用腹部的裂口抓取你。看到他弯腰向前时向后翻滚两次。 |

## 推荐配装

### 削韧流(最推荐)

- **武器:** **巨人碎颅锤**、**接肢大剑**、**石像鬼黑剑**(带出血)或任何+18 以上的特大武器。
- **战灰:** **狮子斩**(单次命中高削韧+高伤害)、**巨兽咆哮**(远程削韧)。
- **护符:**
  - **利爪护符**(跳跃攻击伤害+15%)
  - **青铜龙龟护符+1**(削韧+27.5%)
  - **碧绿龙印护符+1**(耐力回复+11/秒)
  - **龙印大盾护符**(物理减伤+10%)
- **增益:** **爆筋香薰**+**卢恩弧**。

跳跃重击循环:R1 起手 → 跳跃 R2 → 落地立刻再跳跃 R2。约 5-6 次循环触发削韧,打出致命一击后重复。

### 黑焰流(速杀)

- **武器:** **黑焰刀**(战技自带黑焰)或任何武器装备**黑焰之刃**战灰。
- **护符:** 火蝎护符、亚历山大碎片、利爪护符、龙印大盾护符。
- **增益:** **火耐丹**(他的黑焰火球会被自己的黑焰抵消)。

黑焰对神肌造成**真实伤害百分比**:每次命中削减最大 HP 的 2%,持续 3 秒。配合跳跃攻击,约 8-10 次命中就能击败他。

### 通用建议

- **遗灰:** **拟态之泪+10**(分担仇恨)或**黑刀缇切**(高机动性+削韧)。
- **血瓶分配:** 7 红 3 蓝。神肌贵族的单次伤害不算特别高,但连段伤害累积快。
- **盔甲:** 任何中等护甲即可。他的攻击无法靠盔甲硬抗——闪避优先级高于防御。

## 剧情细节

神肌使徒与神肌贵族是游戏中最古老的刺客组织之一,专门猎杀半神并将他们的皮肤剥下制成"神肌"外衣。艾格雷神殿中的这位贵族守护着通往火山官邸的秘密通道——他的存在暗示了神肌一族与拉卡德的联盟关系。

打败他后,从祭坛后方的升降梯下行,可直接进入火山官邸的密道,跳过整个盖利德山脉的迂回路线。这是进入拉卡德区域的最快捷径。

> *神肌贵族的胖不是弱点,而是武器。但所有武器都有破绽——你只需要找到翻滚路径的空隙。*

## 战后路线

击败神肌贵族后,你有两个选择:

1. **继续前往火山官邸**——从祭坛后方下行,进入密道。这会跳过塔妮丝的邀请任务线,但能立刻见到拉卡德。
2. **返回盖利德山脉**——如果你还未接受塔妮丝的邀请信,可以先去完成火山官邸的刺客任务线,再挑战拉卡德。

无论选择哪条路,艾格雷神殿的赐福点是后续传送回盖利德山脉的关键节点——记得激活。
