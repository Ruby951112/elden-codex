## 阶段策略

刺穿王梅瑟莫是玛丽卡隐藏的儿子,也是影之地最具技术性的战斗之一。这场战斗的核心教学是**距离管理**——梅瑟莫的火焰矛覆盖中距离,巨蛇统治近距离,而他的投掷技能惩罚远距离。你必须在这三个区域之间不断切换,根据他的招式前摇做出判断。

> **关键提示:** 梅瑟莫的黑火伤害极高且无法被普通火耐完全抵消。装备**龙印大盾护符**和**焰之守护干腌肝**可将伤害降低约 40%。但真正的答案是:不要被打到。

### 第一阶段——火焰之子

第一阶段的梅瑟莫挥舞一柄燃烧的黑火长矛,攻击节奏缓慢但判定极其精准。他的连段通常是 2-3 段,但每段之间有明显的延迟陷阱。许多玩家在第二段和第三段之间过早翻滚而被抓。

需要记忆的核心攻击:

- **双段矛刺**——第一段向前突进,第二段向上挑。向他右侧(你的左手边)翻滚,**在第二段动作开始后**才翻,不是之前。
- **火焰矛投掷(单发)**——他将矛向后举起 1.5 秒后投出。这是诱骗翻滚的陷阱。等到矛**离手**的瞬间再翻滚。
- **黑火爆发**——矛插入地面后爆炸。爆炸范围比视觉效果大 30%。看到插地动作立刻向后冲刺,不要翻滚——翻滚会让你停在爆炸边缘。
- **旋风斩**——他会旋转 2-3 圈,矛尖划出火焰轨迹。向他背后贴身翻入,旋转内圈是安全的。

血量低于 70% 后,梅瑟莫会开始使用**三连投矛**——连续投出三支火矛。第一支和第二支间隔 0.8 秒,第二支和第三支间隔 1.2 秒。节奏是:翻滚-停顿-翻滚-停顿-翻滚。不要连按闪避。

### 第二阶段——巨蛇之王

血量约 40% 时,梅瑟莫会撕下封印自己的布条,召唤出体内的**深渊巨蛇**。动画持续约 5 秒——这是你喝瓶和上 Buff 的窗口。

**第二阶段的革命性变化:巨蛇有独立的攻击判定**。梅瑟莫的矛攻击和巨蛇的咬击可以同时释放,形成双重压制。你必须时刻追踪两个目标的动作。

核心规则:**永远不要站在蛇头正前方**。巨蛇的咬击速度比梅瑟莫的矛快 40%,且伤害更高。正确站位是梅瑟莫的 4-5 点钟方向(右后侧)——这个位置能同时躲开矛的横扫和蛇的前咬。

第二阶段新增攻击:

- **蛇头俯冲**——巨蛇向前猛咬,覆盖 10 米直线。看到蛇张嘴立刻向左右翻滚,**绝不向后**。
- **火焰吐息锥**——蛇口喷出黑火锥形攻击。垂直蛇头方向冲刺 2 秒,火焰结束后翻入梅瑟莫侧身。
- **双重突刺**——梅瑟莫突刺的同时,蛇从另一侧咬击。这是最致命的组合技。看到梅瑟莫和蛇同时抬手,立刻向**梅瑟莫的背后**翻滚——那是唯一的安全区。
- **爆发环**——梅瑟莫跳起砸地,制造 360° 黑火冲击波。看到跳跃向后翻两次,不是一次。

## 招式与应对

| 招式 | 应对 |
|------|------|
| 双段矛刺 | 第二段**开始后**向他右侧翻入。打两下 R1。 |
| 单发投矛 | 等矛**离手**再翻。过早翻滚会被矛尾判定扫到。 |
| 三连投矛 | 节奏:翻-停-翻-停-翻。不要连按。 |
| 黑火插地爆发 | 向后冲刺,不翻滚。爆炸延迟 0.5 秒。 |
| 旋风斩(2-3 圈) | 向他背后贴身翻入。最后一圈后打一段。 |
| 蛇头俯冲(P2) | 看到张嘴向左右翻。向后翻会被追咬。 |
| 火焰吐息锥(P2) | 垂直蛇头冲刺。锥形持续 3 秒。 |
| 双重突刺(P2) | 向**梅瑟莫背后**翻滚,唯一安全区。 |
| 爆发环(P2) | 跳起瞬间向后翻**两次**。冲击波半径约 8 米。 |

## 推荐配装

梅瑟莫的弱点是**冰霜**和**雷电**。利用这两个属性可显著缩短战斗时间。

- **武器推荐:**
  - **龙枪**或**雷电长牙**(雷属性戳刺武器,攻击后可保持距离)
  - **冰雾**(战技)+ 任意快速武器(触发冻伤后伤害+20%)
  - **龙王岩剑**(冰属性跳劈,第二阶段打蛇头效果拔群)

- **遗灰:** **黑刀缇切+10** 或 **古遗迹骑士奥蕾莉亚**。两者都有良好的机动性和生存能力。**不要**用巨型坦克召唤——梅瑟莫的范围攻击会同时打到你和召唤物。

- **护符:**
  - **龙印大盾护符**(物理减伤 10%)
  - **闪烁的焰之守护护符**(黑火减伤 15%)
  - **深红琥珀勋章+3**(HP+10%)
  - **碧绿龟壳护符**(耐力回复,关键——梅瑟莫惩罚无耐翻滚)

- **圣杯瓶配置:**
  - **焰之守护干腌肝**(黑火减伤 35%,持续 60 秒)
  - **雷电增幅干腌肝**(如果使用雷武器)
  - **冰冻增幅干腌肝**(如果使用冰武器)

- **防具:** 优先**强韧值**和**火耐**。推荐**牛角套装**(高强韧)或**火焰僧侣套装**(高火耐)。

## 第二阶段的锁定管理

这是许多玩家失败的隐藏原因:**锁定目标**。第二阶段如果一直锁定梅瑟莫,你的视角会完全丢失巨蛇的位置。推荐策略:

1. **解除锁定**,自由视角追踪两个目标。
2. 只在攻击窗口短暂锁定梅瑟莫,打完立刻解锁。
3. 习惯用右摇杆手动调整视角——这是高难度 Boss 的必备技能。

如果你不适应自由视角,至少在看到**双重突刺**的前摇时解锁一次,确认蛇的位置。

## 冰霜与雷电的协同

梅瑟莫的双弱点设计不是偶然。**冻伤**状态会让他的防御降低 20%,此时切换到雷电武器能打出爆炸性伤害。推荐流程:

1. 开局用冰雾战技或冰壶触发冻伤(约需 3-4 次命中)。
2. 切换到雷电武器,趁冻伤持续期(约 15 秒)疯狂输出。
3. 冻伤结束后重新积累冰霜,循环往复。

这种玩法在第一阶段能提前 30% 时间触发阶段转换,显著降低整体难度。

> **关于剧情:** 梅瑟莫是玛丽卡派往影之地执行"屠戮"任务的隐子,被巨蛇诅咒且终生背负罪孽。他的台词"母亲……您为何抛弃我?"是 DLC 最悲剧的时刻之一。打他之前先与霍拉·露对话,能解锁完整的剧情线。
