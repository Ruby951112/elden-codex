## 阶段策略

影树化身是阴影之地主线的最终守门人,也是你在挑战最后的大敌前必须跨越的关卡。这场战斗是一次节奏测试——它考验的不是你的反应速度,而是你的耐心与判断力。化身的每一次攻击都有长达1-2秒的明显前摇,但攻击范围和伤害都高得可怕。提前翻滚是大多数玩家阵亡的主因。

竞技场是影树之巅的圆形平台,视野开阔但没有任何掩体。化身本体巨大如黄金树化身,但它的攻击模式更加凶暴,连段更长,且附带暗金色的圣光效果。

> **关键提示:** 影树化身只有**单阶段**,但血量池深达22000——是游戏中最厚的血条之一。这不是速战速决的战斗。做好长期拉锯的准备,切勿因血量下降缓慢而焦躁贪刀。

### 核心机制——根须与圣光

化身的攻击分为三类:

- **树根重击**——用粗壮的根臂砸击地面,范围巨大但前摇明显
- **根须扫击**——横向或纵向的扫击,覆盖半个场地
- **圣光爆发**——身体周围或远程释放暗金色冲击波,附带击飞效果

**最大的陷阱是它的攻击延迟。** 化身会抬起根臂悬停2-3秒,诱骗你提前翻滚。你必须等到根臂开始向下加速的瞬间再翻滚——这个时机比你预期的要晚半秒以上。许多老玩家会在这里翻车,因为肌肉记忆已经习惯了更快节奏的boss。

血量低于40%后,化身会开始使用**三连根刺**——从地面刺出三道根须,每道间隔约1秒。这是它最危险的招式,因为判定持续时间长且难以预判方向。

## 招式与应对

| 招式 | 应对 |
|------|------|
| 双臂下砸 | 等它开始向下加速时向左右翻滚。砸击瞬间贴近它的根部是安全区。打2-3下R1。 |
| 横扫(左至右) | 向它翻入,不要向后。扫击判定高于你的头顶,翻入后绕到背后。 |
| 纵劈(头顶向前) | 左右翻滚。冲击波范围比动画效果更远——不要站在正前方10米内。 |
| 圣光环(近身) | 它蓄力时身体会发光。向后翻滚两次。第二次翻滚清出爆炸半径。 |
| 根须投射 | 从地面远程刺出根须。看到地面金光闪烁立刻侧向冲刺。不要翻滚——持续时间太长。 |
| 三连根刺(血量<40%) | 第一道根须出现时向化身方向冲刺。贴近它的根部是最安全的位置。 |
| 升天圣光 | 它向上飘浮,全场AOE爆发。垂直它的方向全力冲刺3秒,在爆发前逃出半径。 |

## 推荐配装

影树化身对火焰和圣属性伤害有显著弱点。以下配装能大幅缩短战斗时间:

- **武器:** 
  - **火焰骑士大剑+25**(火焰战技"火焰冲锋"伤害爆炸)
  - **黄金律法剑+10**(圣属性武器中最稳定的选择)
  - **火焰喷射祷告**(法师流最优解——持续输出且安全距离远)
- **遗灰:** 
  - **古龙骑士克里斯朵夫+10**(坦度极高,能持续吸引化身注意力)
  - **欧嘉+10**(远程弓箭输出,几乎不会被秒)
- **护符:**
  - **龙印大盾护符+2**(物理减伤20%)
  - **圣伤护符+3**(圣属性防御——化身的圣光爆发是主要威胁)
  - **深红琥珀勋章+3**(HP提升——容错率提升显著)
  - **金角护符+2**(强韧——避免被根须擦伤打断翻滚)
- **圣杯瓶配置:**
  - **圣力防护结晶泪**(减免一次圣伤——在血量<40%时喝下)
  - **强韧结晶泪**(短时间内不会被击飞——适合近战抢输出窗口)
- **道具:** **火焰壶×10**、**火油脂**。即使你是纯近战,远程火焰壶也能在安全输出窗口不足时补刀。

> **装备前置:** 如果你的**影树祝福等级**低于+12,这场战斗会非常吃力。化身的伤害设计假设你已经收集了大部分影树碎片。回去把悬崖、洞窟、教堂的碎片全部收集完再来——这不是选择题。

## 站位与输出窗口

化身的攻击虽然前摇长,但输出窗口非常短。以下是最安全的输出时机:

1. **双臂下砸后** — 翻滚到它的根部后方,打2-3下R1。它抬起根臂前撤离。
2. **横扫后** — 如果你成功翻入它背后,有约4秒的自由输出时间。这是最长的窗口。
3. **圣光环后** — 它释放完毕会有短暂僵直。冲刺贴近打1-2下即可。
4. **三连根刺时** — 如果你贴在它根部,这段时间它完全无法攻击你。全力输出。

**绝对不要贪刀的时机:**
- 它开始蓄力圣光(身体发光)时还在它脚下
- 横扫动画结束但你在它正面
- 血量低于20%时——它会无预兆接大范围AOE

## 心态与节奏

影树化身是一场**马拉松,不是冲刺**。22000血量意味着即使你完美执行,战斗也会持续8-12分钟。许多玩家在第5-6分钟时因焦躁而失误——他们看到化身血量缓慢下降,开始贪刀或提前翻滚。

正确的心态是:

- 每次只打一段连招。
- 打完立刻拉开距离。
- 不要试图"加速"战斗——这只会导致重开。

化身的攻击模式没有随机性。它的每一次抬手都对应固定的后续招式。当你开始能预判它的下一个动作时,战斗就已经赢了一半。

> *影树化身教你的不是技巧,而是纪律。* 如果你能不贪刀地打完这场战斗,后面的真正终战会比你想象的简单。

## 战后与剧情

击败影树化身后,你将获得**32万卢恩**——DLC中最高的单次掉落。更重要的是,前方的道路会向你开启,通往阴影之地的最终秘密。

化身的存在本身是个谜。它守护的不是影树本身,而是影树之下的某个入口。玛莉卡在这里封印了什么?为什么守卫的形态如此接近黄金树化身?这些答案在后续战斗中会逐步揭晓。
