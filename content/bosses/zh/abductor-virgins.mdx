## 阶段策略

拐少女是火山官邸的双人组 Boss——两台巨型机械同时向你进攻。这场战斗的设计哲学很明确:**二打一必输,一打一必赢**。你的首要目标不是伤害,而是**拆分战场**。

> **关键提示:** 雾门后方有一条长廊。引诱其中一台拐少女留在走廊内,另一台自然会进入内室。利用墙壁和空间差分别击破。千万不要在内室同时对抗两台——那是为召唤玩家设计的难度。

### 唯一阶段——机械双子

拐少女没有阶段转换,但有两种形态:

- **轮形态**——卷成球体滚动攻击,移动速度极快
- **人形态**——站立状态,使用抓取、横扫、火焰吐息

核心理解:拐少女在**轮形态下无敌**。所有攻击都会被弹反。你必须等它们展开成人形,才有输出窗口。

战斗流程:

1. 进入雾门后**立刻后退**到走廊中段
2. 锁定较近的一台,攻击一次引诱它追击
3. 另一台通常会卡在走廊尽头或进入内室——利用这个距离差
4. 集火击杀第一台(约 2 分钟)
5. 清理第二台时已是标准单挑

拐少女的 AI 有个致命缺陷:**它们不会同时变形**。如果一台展开成人形,另一台往往会保持轮形巡逻。你有完整的 5-8 秒窗口单独输出一台。

## 招式与应对

| 招式 | 应对 |
|------|------|
| 滚动冲锋(轮形态) | 向左侧翻滚。不要向后翻——追踪性极强。 |
| 双臂抓取(人形态) | 看到双手张开向前翻滚穿过它。被抓住会受到致命伤害+强制传送到火山地下。 |
| 横扫砸地 | 它会双手砸向地面并横扫 180°。贴身站在它正后方是绝对安全区。 |
| 火焰吐息 | 胸腔打开喷射锥形火焰。垂直它的方向冲刺,锥形覆盖范围比视觉效果窄。 |
| 旋转斩 | 双臂展开旋转 3 圈。向后拉开距离,不要贪刀。 |
| 协同滚动(双人) | 两台同时变成轮形夹击。这是你分而治之失败的信号——立刻撤回走廊重新拆分。 |

> **关于抓取:** 拐少女的抓取判定异常宽泛,且被抓后会强制传送到火山地下深处。虽然那里有独特战利品(祈祷者套装、锻造石),但大多数玩家希望先打完火山官邸再探索。**避免被抓的唯一方法是向前翻滚**——侧翻和后翻都不够快。

## 推荐配装

拐少女是机械敌人,因此吃雷电与出血双重弱点:

- **武器:** 
  - **雷切打刀+雷油脂**——出血+雷电双重触发
  - **红狮子大弓+雷箭**——远程安全输出
  - **黄金戟+雷矛**——信仰流标准答案
- **战灰:** **猎兽咆哮**对机械敌人额外伤害+20%。可以用在任何武器上。
- **护符:**
  - **避雷干肝**——减少火焰吐息伤害
  - **龙印大盾护符**——减少物理伤害
  - **斧护符**——增加蓄力攻击伤害(对轮形态无效,但人形态弱点是头部)
- **遗灰:** **结晶人骨灰+6** 或更高。结晶人的远程攻击能分散目标优先级,且不会像近战召唤一样被滚动秒杀。

如果你是法师流:**岩石投射**与**陨石**能在轮形态下命中它们的核心,略过无敌判定。这是少数几场魔法流比近战更有优势的战斗。

## 替代策略——纯远程

如果你实在无法掌握闪避时机,火山官邸提供了一个取巧方案:

1. 进入雾门后立刻跑回走廊尽头
2. 使用**长弓+羽箭**或**洛蕾塔的大弓**(魔法)
3. 拐少女在滚动冲锋到走廊尽头时会卡住约 2 秒——利用这个窗口射击
4. 战斗时长约 8-12 分钟,但几乎零风险

这不是最优雅的方法,但对卡关的玩家来说绝对有效。FromSoftware 在设计上允许这种策略——走廊的长度就是线索。

## 战利品与后续

击败拐少女后获得:

- **24000 卢恩**
- **探查者套装**(可以从尸体拾取)
- **火山官邸继续向前的通道**

如果你**被拐少女抓取传送**到了火山地下,别慌:

1. 你会出现在火山地下熔岩区域
2. 向北走找到赐福**火山地下洞穴**
3. 这里可以收集**大量锻造石**和**祈祷者套装**
4. 最深处有一台单独的拐少女——击败后传送回火山官邸走廊

许多玩家**故意**让自己被抓一次,先刷完地下再回来正常击败双子。这是完全可行的路线选择。

> *拐少女教你的是空间管理。* 在狭窄走廊击败它们,和在开阔内室同时对抗两台,是完全不同的两场战斗。学会利用地形,后面的双人 Boss(例如双王子、双树哨兵)会简单很多。

## 相关 Boss

火山官邸的真正考验不是拐少女,而是更深处的**蛇人诺尔**——他才是火山支线的核心。击败拐少女后,继续向前探索官邸二层,找到**坦尼斯**并推进火山任务线。
